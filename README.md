# EcoVision — Мониторинг экосистемы с использованием предобученной модели ML

## Описание проекта

Мониторинг экосистем требует регулярного анализа данных с фотоловушек. 
Ручная обработка изображений является трудоёмкой, занимает значительное время и может приводить к ошибкам при выявлении критических событий (например, присутствие человека в охраняемой зоне).

Цель проекта — разработать прототип IT-системы для автоматического анализа изображений с использованием предобученной модели машинного обучения.

Проект демонстрирует, как технологии компьютерного зрения могут быть использованы для:
- автоматической детекции объектов на изображениях,
- формирования статистики,
- выявления потенциально критических событий,
- поддержки принятия решений в сфере экологии.

---

## Используемые технологии

- Python
- Ultralytics YOLOv8 (предобученная модель)
- OpenCV
- Pandas
- Matplotlib
- Google Colab

Модель используется без дообучения (inference-only).

---

## Данные

Использовано 20 изображений с фотоловушек:
- ночные ИК-кадры
- дневные сцены
- изображения с животными
- пустые кадры
- изображения с человеком

---

## Логика работы системы

1. Загрузка и распаковка изображений.
2. Инференс предобученной модели YOLOv8.
3. Сохранение детекций в `detections.csv`.
4. Сохранение изображений с bounding boxes.
5. Расчёт количественных метрик.
6. Визуализация распределений.
7. Формирование алертов:
   - **CRITICAL** — обнаружение человека с высокой уверенностью
   - **INFO** — обнаружение человека с низкой уверенностью
   - **WARNING** — высокая доля пустых кадров

---

## Количественная оценка (без ручной разметки)

- Обработано изображений: 20
- Пустых кадров: 6 (30%)
- Всего детекций: 19
- Средняя confidence: ~0.48
- Медианная confidence: ~0.45

---

## Ограничения

- Используется модель общего назначения (COCO), отсутствуют специализированные классы (например, wolf).
- Возможны ложные срабатывания на ночных ИК-кадрах.
- Отсутствует ручная разметка → невозможно рассчитать precision/recall.
